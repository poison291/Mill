---

## Overview

Mill enables interactive, multi-turn conversations with context memory. It leverages a Go backend to handle communication with locally stored LLMs and a modern React + Tailwind frontend for a clean, user-friendly interface.

The system is designed to prioritize privacy and independence, allowing users to interact with AI models without relying on cloud services or external accounts.

---

## Key Features

- **Offline-first architecture:** Fully functional without an internet connection once models are installed locally.  
- **Multi-turn conversation:** Maintains context across multiple messages for coherent interactions.  
- **Flexible integration:** Designed to easily add features like local data access, workflow automation, or multi-user LAN setups.  
- **Responsive UI:** Built with React and Tailwind for a clean and intuitive experience.  
- **Modular design:** Backend and frontend are decoupled, allowing independent development and easier future packaging.
